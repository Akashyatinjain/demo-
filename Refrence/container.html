Containers

Container Header Description
std::vector <vector> A dynamic array that can resize itself automatically when elements are added or removed.
std::list <list> A doubly-linked list that allows fast insertion and deletion of elements anywhere in the sequence.
std::deque <deque> A double-ended queue that supports fast insertion and deletion at both the beginning and the end.
std::array <array> A fixed-size array container that provides a safer and more convenient interface compared to raw arrays.
std::set <set> A sorted associative container that contains unique elements.
std::map <map> A sorted associative container that stores key-value pairs with unique keys.
std::unordered_set <unordered_set> An unordered associative container that contains unique elements.
std::unordered_map <unordered_map> An unordered associative container that stores key-value pairs with unique keys.
std::stack <stack> A container adapter that provides a LIFO (Last-In-First-Out) data structure.
std::queue <queue> A container adapter that provides a FIFO (First-In-First-Out) data structure.
std::priority_queue <queue> A container adapter that provides a priority queue, where elements are ordered by priority.
