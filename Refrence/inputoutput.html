Input/Output Stream Library

Stream Classes

The C++ Standard Library provides functionality to use an abstraction called streams specially designed to perform input and output operations on sequences of character, like files or strings.

This functionality is provided through several related classes, as shown in the following relationship map, with the corresponding header file names on top:

Standard Stream Objects

Object Type Description
cin istream Standard input stream (usually keyboard)
cout ostream Standard output stream (usually display)
cerr ostream Standard error stream (unbuffered)
clog ostream Standard logging stream (buffered)

Common I/O Operations

Operation Description Example
<< (insertion) Outputs data to a stream cout << "Hello";
>> (extraction) Inputs data from a stream cin >> variable;
get() Reads a single character ch = cin.get();
getline() Reads a line of text getline(cin, str);
